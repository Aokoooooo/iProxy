## How to contribute

### dev

```shell
git clone https://github.com/alibaba/lightproxy
cd lightproxy
yarn run install-deps
yarn run dev
```

For new contributors you can try to fix a [good-first-issue](https://github.com/alibaba/lightproxy/labels/good%20first%20issue)
